# 1️⃣ Conditional Statements (`if`, `elif`, `else`)

Conditional statements are used for decision making.

## Example

```python id="a1l7bz"
age = 18

if age >= 18:
    print("You can vote")
else:
    print("You cannot vote")
```

# 2️⃣ Loops

Loops help repeat tasks automatically.

---

## 🔁 `for` Loop

```python id="hh0w3m"
for i in range(5):
    print(i)
```

---

## 🔄 `while` Loop

```python id="g6hh0m"
count = 1

while count <= 5:
    print(count)
    count += 1
```

---

## Learn

- `range()`
- Nested loops
- Infinite loop concept
- `break`
